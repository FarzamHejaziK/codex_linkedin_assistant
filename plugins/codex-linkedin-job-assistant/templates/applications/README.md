# applications/

Codex creates one folder per application or referral-material package:

```text
<YYYY-MM-DD>_<Company>_<Role>/
```

Typical contents:

- `job_description.md`
- `resume_source.<ext>`
- `resume.pdf`
- `cover_letter.md`
- `notes.md`
- `contacts.md`

This folder is private job-search state and is ignored by default.
