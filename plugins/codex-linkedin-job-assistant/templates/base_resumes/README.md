# base_resumes/

Optional folder for editable resume sources or templates.

Examples:

- `resume.tex`
- `resume.docx`
- `resume.md`
- `resume.html`

Editable sources allow Codex to tailor a role-specific version in `applications/<date>_<company>_<role>/`. If you only use a PDF, Codex can attach it but cannot truly tailor it.
