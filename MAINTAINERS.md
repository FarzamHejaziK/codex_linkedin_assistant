## Approval Model

This is a single-maintainer project. All PRs require [@FarzamHejaziK](https://github.com/FarzamHejaziK)'s approval before merge, enforced via [`.github/CODEOWNERS`](.github/CODEOWNERS) plus branch protection on `main`.

Direct pushes to `main` should be blocked on the public project. Force-push and branch deletion should be blocked.

## Becoming a Maintainer

Out of scope for v0. If the project grows enough that a co-maintainer is justified, we will add a `GOVERNANCE.md` covering the process. For now: contribute via PRs, build a track record, and we will talk if it is a good fit.
