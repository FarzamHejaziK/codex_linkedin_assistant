# resumes/

Put your resume files here. Supported formats:

- `.pdf`
- `.tex`
- `.docx`
- `.md`
- `.html`

Codex reads these files to infer your name, current role, target roles, skills, and a short profile summary.

## Optional Search Profile

Create `search_profile.md` if you want to guide job discovery beyond your resume.

Example topics:

- Must-have locations
- Remote/onsite preferences
- Salary floor
- Role level
- Industries of interest
- Deal-breakers
- Company size or stage preferences

The search profile describes what you want. Your resume describes what you can do. If they conflict, the search profile wins.
